== Installation ==

1. Navigate to `Plugins > Add New` within your WordPress admin.
2. Search for **"WP Adminify"**.
3. Click **"Install Now"** and then **"Activate"**.
4. A new menu item **"WP Adminify"** will appear. Click on it to start customizing!
5. (Optional) For even more power, [upgrade to Pro](https://wpadminify.com/pricing/) to unlock all addons and features.

== Frequently Asked Questions ==

= Can I use the white label features for free? =
Yes! The free version includes robust white-labeling options. The Pro version unlocks even more advanced branding controls.

= Will this plugin slow down my website? =
No. WP Adminify is built for performance. It only loads its scripts on the WordPress admin area, leaving your front-end site speed completely unaffected.

= Is it compatible with my theme and other plugins? =
Absolutely. WP Adminify is designed to be compatible with all standard WordPress themes and most popular plugins.

= Do I need coding skills to use it? =
Not at all. WP Adminify provides an intuitive, point-and-click interface. No code required!
